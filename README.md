Welcome to the repository for the [Globus Compliance Portal](https://compliance.globus.org/). This site hosts public policies, security documentation, and resources designed to support institutional compliance, privacy, and data stewardship requirements.

## Overview

Globus is built to ensure secure, reliable research data management across institutions. The platform incorporates strong authentication, encrypted data transfer, granular access controls, and comprehensive audit logging to help institutions meet internal policies and external regulatory obligations.

This portal is build using [Globus's data portal template](https://github.com/globus/template-data-portal).
